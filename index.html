Sticky and Normal Drop-Down Menu Header

What I hope to practice/learn?

Since I started this bachelor I find that the navigational headers are sometimes really tricky, specially when you try to make them stylish. I hope to learn better techniques to improve my navigation headers styles and looks. Also, I wish to practice my floats and clear coding and to get a better understanding of this tags.

How this relates to what I have/will learn in WDD?

Having a menu helps the user to navigate the website and find what they are looking for without too much hassle. This is something that I associate with the design classes that I have taken so far where having a design that helps the user navigate is proficient and brings accessibility to the web page. I want to add the flexibility for the user to have the navigational header appear always in a static mode so they don't have to scroll all the way up to go to a specific part of the web page.

How might I build this?

  1. Create the component as a <header>
  2. Add a <div> to contain the <nav> elements.
  3. Company logo using <img> or <h1>
  4. Add a <a> tag to make the logo a link to the main page.
  5. Navigation <nav> containing a <ul> with five <li>, each containing an <a>
  6. Dropdown menu items will be nested <ul> inside of corresponding <li>
  7. To make it stick to the top I can either use jQuery or using CSS position fixed
    and add some animation to make it appear like in the example.
